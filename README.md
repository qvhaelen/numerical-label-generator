# numerical-label-generator
It is a GUI to generate labels made of numbers and units. The GUI allows the user to  select various types of properties and effects to customize the labels.

The user should create a new virtual  environment "env_name" with the following requirements:
PyQt6=6.9.1
matplotlib=3.7.0
scipy=1.11.0
numpy=1.24.0
python 3.11

The library pillow is also required

Once the environment is created, activate the environment using conda activate env_name

One can call the GUI using the following command:

python label_generator_gui.py
